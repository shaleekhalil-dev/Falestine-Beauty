# Falestine Beauty Portfolio 🌸
**Hybrid Strategic Leadership in Beauty & Digital Excellence**

A high-end professional portfolio developed for **Falestine Karakra**, a skincare and beauty specialist. This project blends aesthetic elegance with modern web technologies to reflect luxury and clinical precision.

## ✨ Key Features
- **Glassmorphism UI:** A modern, translucent interface designed to align with the premium beauty industry identity.
- **Smart Booking System:** Fully integrated with **Google Sheets** via API to manage appointments and prevent scheduling conflicts programmatically.
- **Optimized Media Gallery:** Structured display for introductory videos and "Magic Touches" using responsive grid systems.
- **Bilingual Support (i18n):** Seamlessly toggles between **Arabic** and **English** with full **RTL/LTR** layout adaptation.
- **Toast Notifications:** Smooth, non-intrusive UI feedback for an enhanced user experience.

## 🛠 Tech Stack
- **Frontend:** HTML5, CSS3 (Advanced Flexbox & Grid), JavaScript (ES6+).
- **Backend/Integration:** Google Apps Script, Google Sheets API.
- **Design:** Font Awesome Icons, Custom Glassmorphism Styles.
- **Deployment:** GitHub & Vercel.

## 📸 Project Structure
- `assets/`: Organized directory for optimized images and video assets.
- `css/`: Modular stylesheets focusing on visual identity and responsiveness.
- `js/`: Core logic for translation, UI interactions, and database synchronization.
